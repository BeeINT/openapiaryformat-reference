openapiaryformat reference |version|
============================================


This format creates a way for different independent projects and data sources to share their data of bees and apiaries. 
The format specifications and reference implementations for an easy start will be public available and open source.

You can get your version of this document at http://openapiaryformat-reference.readthedocs.org
